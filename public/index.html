📱 想每天在微信收到打卡提醒？

  1. 访问 sct.ftqq.com 扫码登录，获取 SendKey
  2. Fork 此仓库到你的 GitHub
  3. 在仓库 Settings → Secrets and variables → Actions → 添加 SENDKEY
  4. 在 Actions 中启用 Workflow
  5. 每天 6:05、9:30、18:05、21:20 自动发微信提醒 🎉

✅ 提醒完全自动运行，无需开电脑、无需部署服务器！
